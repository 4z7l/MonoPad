# MonoPad


## Movie + NotePad = MonoPad


## Specification
- Architecture : MVVM
- Network : Retrofit2, OkHttp3
- Image : Glide
- Auth : Firebase-auth
- Dependency Injection : Koin
- Internal DB : Room
- Async : RxJava
- Jetpack (Databinding, LiveData, ViewModel)
- Other : ViewPager2, Cardview, RecyclerView..etc
